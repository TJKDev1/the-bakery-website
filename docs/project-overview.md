# Project Overview

## About The Bakery

The Bakery is a meme and humor merchandise brand that sells products through Redbubble. The tagline is **"We Don't Cook We Bake!"** - a play on words combining bakery themes with internet slang about creating content.

## Brand Identity

| Element | Value |
|---------|-------|
| **Name** | The Bakery |
| **Tagline** | We Don't Cook We Bake! |
| **Established** | 2026 |
| **Theme** | Memes, jokes, and humor |
| **Platform** | Redbubble |

## Website Purpose

This website serves as:

1. **Landing page** - First impression for visitors
2. **Brand showcase** - Displays the brand identity and values
3. **Shop gateway** - Links to Redbubble product categories
4. **Contact hub** - Email and social media links

## Target Audience

- Meme enthusiasts
- People looking for funny merchandise
- Gift buyers seeking humorous items
- Social media users (TikTok audience)
